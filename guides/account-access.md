# Account access

Use [app.stacklabs.bio](https://app.stacklabs.bio/) for Stack Labs account access.

## Create an account

Open the App and choose the available registration action. Enter an email address you can access and follow the confirmation message when one is required.

## Sign in

Use the same email address and password you used when creating the account. If the App reports that the credentials are not accepted, check the email address for typing errors before trying again.

## Recover access

Use the password-recovery action on the sign-in screen. Open only the most recent recovery message and complete the flow in the browser where you want to use Stack Labs.

## Keep account information private

Stack Labs support should never need your password, email confirmation code, password-recovery code, or authenticated session link. Do not include those values when reporting a problem.
