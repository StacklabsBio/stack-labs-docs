# Getting started

Start with the [Stack Labs website](https://stacklabs.bio/). The same website is also available at [www.stacklabs.bio](https://www.stacklabs.bio/).

The website includes a direct link to the Stack Labs PWA. Selecting the app call to action opens [app.stacklabs.bio](https://app.stacklabs.bio/), where you can sign in or begin the available beta flow.

The PWA runs in a browser and can be installed on supported devices. Browser and operating-system install behavior varies, so follow the prompt offered by your device when it is available.

Stack Labs is currently a beta product. Use the current interface as the source for available actions and account options.
