# Getting started

Start with the [Stack Labs website](https://stacklabs.bio/). It introduces the product and provides a direct route into the App.

## 1. Open the App

Open [app.stacklabs.bio](https://app.stacklabs.bio/). Create an account or sign in with the account you already use for Stack Labs.

## 2. Choose how you use it

The App works in a browser. On supported devices, you can also [install the App](/guides/install-the-app) for quicker access from your Home Screen or desktop.

## 3. Start with one daily action

After signing in, review your current stack or complete today's check-in. The App keeps the full product experience in one place; the Telegram companion provides a lighter route to selected daily actions.

If you cannot reach your account, use the [account access guide](/guides/account-access). Never send a password or authentication code to support.

Stack Labs is under active development. Use the current interface as the source for available actions and account options.
