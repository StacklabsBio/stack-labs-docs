# Using the app

After signing in, the App brings the main Stack Labs activities into one place.

## Home and daily check-in

Home shows the actions and summaries available for the current day. A daily check-in records the current supported measures and returns you to the updated Home view.

## Stacks

Use Stacks to create and review the protocols you are tracking. Follow the labels in the current App when adding, editing, ending, or reviewing a stack.

## Labs and insights

Labs provides a private workflow for supported records and review steps. Insights presents informational summaries based on available product data. These features are designed to help with organization and reflection; they are not medical advice and do not replace professional care.

## Profile and account

Profile and account areas contain controls for the current session and available product preferences. If the interface changes, follow the labels and guidance shown in the current App.

For selected daily actions outside the full App, see the [Telegram companion guide](/guides/telegram-companion).
