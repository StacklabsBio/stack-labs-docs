# Install the App

Stack Labs is a progressive web app. It runs in a browser and can be added to supported phones and desktops for app-like access.

## iPhone or iPad

1. Open [app.stacklabs.bio](https://app.stacklabs.bio/) in Safari.
2. Open Safari's Share menu.
3. Choose **Add to Home Screen** and confirm.
4. Launch Stack Labs from the new Home Screen icon.

If **Add to Home Screen** is not visible, scroll through the available Share actions or confirm that the page is open in Safari rather than an in-app browser.

## Android

1. Open [app.stacklabs.bio](https://app.stacklabs.bio/) in a supported browser.
2. Use the browser's **Install app** or **Add to Home screen** action when it is available.
3. Confirm the installation and launch Stack Labs from the new icon.

## Desktop

Supported desktop browsers may show an install action in the address bar or browser menu. You can always continue using the App in the browser if installation is unavailable.

Installation labels vary by browser and operating system. Installing the App does not create a second account; sign in with your existing Stack Labs account.
