# Support

For current product information, visit the [Stack Labs website](https://stacklabs.bio/) and use the official contact route presented there. For account access, follow the [account access guide](/guides/account-access) first.

When reporting an issue, describe the page or action you were using and what happened. Do not include passwords, authentication codes, private health information, or other sensitive account data.

Helpful details include the device type, browser, page name, action you selected, and the wording of any visible error. A screenshot can help only when it contains no private account or health-adjacent information.
